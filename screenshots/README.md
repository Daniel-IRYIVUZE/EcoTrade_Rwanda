### Screenshot Guidelines

- **Resolution:** 1920x1080 (Full HD) recommended
- **Format:** PNG for best quality
- **Browser:** Use Chrome or Firefox with full window (not just viewport)
- **State:** Show dashboards with data populated (use demo credentials)
- **Clean UI:** Hide browser dev tools, close unnecessary tabs
- **Consistency:** Use the same browser zoom level (100%) for all screenshots

### How to Capture Screenshots

#### Windows
1. Navigate to the page you want to capture
2. Press `Windows + Shift + S` for Snipping Tool
3. Select "Full-screen snip" or drag to select area
4. Save to this folder with the correct filename

#### macOS
1. Navigate to the page you want to capture
2. Press `Cmd + Shift + 3` for full screen
3. Or `Cmd + Shift + 4` to select area
4. Rename and move to this folder

#### Browser Extensions (Recommended)
- **Fireshot** - Full page screenshots
- **Awesome Screenshot** - Annotations and editing
- **Nimbus Screenshot** - Multi-page capture
